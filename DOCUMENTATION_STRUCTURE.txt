✨ KEY PRINCIPLES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. ONE DOCUMENT = ONE AUDIENCE
   Don't mix human-friendly + AI-detailed in same doc

2. NEVER DUPLICATE
   Reference other docs instead of repeating

3. KEEP SYNCHRONIZED
   Update docs when code changes

4. SCANNABLE
   Use tables, emojis, clear headers

5. ACTIONABLE
   Every doc should tell you what to do next
