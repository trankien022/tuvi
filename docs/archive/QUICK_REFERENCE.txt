→ Check: https://gayle-hematogenous-enid.ngrok-free.dev/health in browser

Problem: "Port 3000 already in use"
→ Run: netstat -ano | findstr :3000
→ Run: taskkill /PID <PID> /F

Problem: "URL stopped working"

→ Run: npm run mcp:tunnel
→ Get new URL and update ChatGPT

→ Make sure URL starts with https:// (not http://)
